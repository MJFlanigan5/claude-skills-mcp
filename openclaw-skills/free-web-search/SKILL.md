Search the web for free — no API keys, no credits, no built-in tools required.

## Usage

### Search Wikipedia
```bash
web-search "Artificial Intelligence"
```

### Fetch Any Webpage
```bash
web-search "https://example.com/article"
```

### Direct Command
```bash
curl -s "https://r.jina.ai/http://en.wikipedia.org/wiki/Topic"
```

Uses **Jina AI** (r.jina.ai) — a free service that:
- Takes any URL
- Returns clean markdown text
- Removes ads and clutter
- No signup required

Plus **Wikipedia API** for structured knowledge.
